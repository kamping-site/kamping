Detailed Description

template<typename... Args>
class kamping::MPIResult< Args >

MPIResult contains the result of a MPI call wrapped by KaMPIng.

A wrapped MPI call can have multiple different results such as the recv_buffer, recv_counts, recv_displs etc. If the buffers where these results have been written to by the library call have been allocated by/transferred to KaMPIng, the content of the buffers can be extracted using extract_<result>. Note that not all below-listed buffer categories need to be used by every wrapped MPI call. If a specific call does not use a buffer category, you have to provide ResultCategoryNotUsed instead.

Template Parameters
ArgsTypes of return data buffers.
