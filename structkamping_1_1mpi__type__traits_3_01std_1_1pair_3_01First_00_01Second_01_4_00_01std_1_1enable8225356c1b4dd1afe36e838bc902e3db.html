Specialization of the mpi_type_traits type trait for std::pair, which represents the pair as a sequence of bytes.

Specialization of the mpi_type_traits type trait for std::pair, which represents the pair as MPI struct type.

Warning
This is UB, because byte serialization is only well-defined for trivially copyable types. For a safe version see kamping/types/utility.hpp.
Note
Using struct types may have performance pitfalls if the types has padding. For maximum performance, use the unsafe version in kamping/types/unsafe/utility.hpp.
