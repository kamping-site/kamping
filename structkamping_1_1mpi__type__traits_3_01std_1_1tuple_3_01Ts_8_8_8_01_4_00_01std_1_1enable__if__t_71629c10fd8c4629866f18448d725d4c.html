Detailed Description

template<typename... Ts>
struct kamping::mpi_type_traits< std::tuple< Ts... >, std::enable_if_t<(has_static_type_v< Ts > &&...)> >

Specialization of the mpi_type_traits type trait for std::tuple, which represents the tuple as MPI struct type.

Specialization of the mpi_type_traits type trait for std::tuple, which represents the tuple as a sequence of bytes.

Note
Using struct types may have performance pitfalls if the types has padding. For maximum performance, use the unsafe version in kamping/types/unsafe/tuple.hpp.
Warning
This is UB, because byte serialization is only well-defined for trivially copyable types. For a safe version see kamping/types/tuple.hpp.
